# OpenClaw Adventure Games on Tenstorrent 🎮⚡

**Three AI-powered text adventure games running on Tenstorrent hardware**

## What This Is

Interactive text adventures where an AI agent (running on Tenstorrent chips) acts as game master, creating dynamic narratives in real-time. Think D&D meets classic text adventures, powered by hardware-accelerated AI.

## The Games

### 🧩 Chip Quest (Educational)
Journey inside a Tenstorrent chip. Battle Memory Grues, solve puzzles about Tensix cores, NoC routing, and parallel processing.
- **Playtime:** 30-45 min
- **Style:** Zork-inspired puzzle adventure
- **Learn:** TT architecture through gameplay

### ⚔️ Terminal Dungeon (Roguelike)
ASCII dungeon crawler with procedural generation. Fight 6 types of TT-Grues, collect legendary artifacts, master tactical combat.
- **Playtime:** 30-60 min (replayable)
- **Style:** NetHack-inspired roguelike
- **Features:** Permadeath, GURPS mechanics, loot

### 🎪 Conference Chaos (Trading Sim)
Navigate EAC 2026 as a trader. Buy low, sell high, corner markets, survive Vogon poetry panels.
- **Playtime:** 30-45 min
- **Style:** Trade Wars meets Hitchhiker's Guide
- **Features:** 5 endings, economic simulation, absurdist humor

## Quick Start

```bash
cd adventure-games/scripts
./adventure-menu.sh
```

Choose your game, then **type numbers** (1, 2, 3, etc.) when presented with choices.

## How It Works

- **You type choices** → OpenClaw Gateway → AI Agent → vLLM on Tenstorrent → LLM generates response
- **Interactive:** Requires player input (not autoplay)
- **Stateful:** Game progress persists across sessions
- **Dynamic:** Every response generated by AI in real-time

## Requirements

- OpenClaw v2026.3.2+
- vLLM on Tenstorrent hardware
- Llama 8B+ model (70B for richer narratives)

## FAQ

**Q: Do the games autoplay?**
A: No - these are **interactive adventures**. You choose what to do by typing numbers.

**Q: How do I make choices?**
A: When you see options like:
```
1. Enter the cave
2. Talk to the NPC
3. Check inventory
```
Just type `1`, `2`, or `3` and press Enter.

**Q: Can I play in a browser?**
A: Currently terminal-only (TUI). The games work great in a terminal with proper formatting, colors, and ASCII art.

**Q: Can I modify games?**
A: Yes! Edit `adventure-games/games/*/SOUL.md` files to change rules or create new content.
