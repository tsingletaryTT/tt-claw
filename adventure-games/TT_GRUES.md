# TT-Grues: Tenstorrent's Legendary Creatures 🐉

**"It is pitch dark. You are likely to be eaten by a tt-grue."**

## Overview

TT-grues are educational creatures inspired by the legendary grue from Zork (1977). Each represents a real hardware/software challenge in Tenstorrent chip architecture and parallel computing.

## The Six TT-Grues

| Grue | Symbol | Found | Teaches |
|------|--------|-------|---------|
| **Memory Grue** | 💾 | Dark DRAM caverns | Memory management, cache optimization |
| **Latency Grue** | 🐉 | Slow NOC corridors | Pipeline optimization, reducing latency |
| **Bandwidth Grue** | 🌊 | Congested pathways | Network optimization, bandwidth management |
| **Stack Grue** | 📚 | Deep recursion | Stack overflow prevention, iteration |
| **Race Condition Grue** | ⚡ | Parallel zones | Thread safety, synchronization |
| **Deadlock Grue** | 🔒 | Resource conflicts | Deadlock prevention, lock ordering |

## How to Defeat Grues

Each grue is defeated by specific Tenstorrent hardware-themed items:

- **Cache Lantern** → Lights up DRAM caverns (defeats Memory Grue)
- **NOC Optimization Scroll** → Speeds up data flow (defeats Latency Grue)
- **Ethernet Warp** → Opens new paths (defeats Bandwidth Grue)
- **Stack Unwind Scroll** → Returns to safe depth (defeats Stack Grue)
- **Mutex Shield** → Prevents race conditions (defeats Race Condition Grue)
- **Lock Ordering Scroll** → Breaks deadlocks (defeats Deadlock Grue)

## Where Grues Appear

- **Chip Quest:** DRAM caverns, slow NOC paths, parallel processing zones
- **Terminal Dungeon:** Deep levels, dark rooms, recursive puzzles
- **Conference Chaos:** Easter egg references only

## Educational Value

Grues make learning fun by:
- Representing real computing challenges
- Teaching proper solutions through gameplay
- Using humor to mask the learning
- Providing instant, funny respawns (no frustration)

## Classic Grue Traditions

- **Warning:** "It is pitch dark. You are likely to be eaten by a tt-grue."
- **Death:** "You have been eaten by a grue. Your adventure ends here."
- **Wisdom:** "A hollow voice says 'Fool'."

All deaths are humorous with instant respawn and a lesson learned!
